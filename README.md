# STF

### Requeriments

- NodeJS
- Bower
- RethinkDB

### Install

- `brew install rethinkdb`
- Make internal npm work
- `brew install protobuf`
- `brew install zmq`
- `npm install -g bower`
- `npm install`
- `bower install`
- `npm link`

### Run

- `rethinkdb`
- `stf local`

### Update

- `git pull`
- `npm install`
- `bower install`
